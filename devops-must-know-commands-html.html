### DevOps Must-Know Command List

#### 1. **Version Control (Git)**
- Initialize Repository:
  git init
- Clone Repository:
  git clone 
- Check Status:
  git status
- Add Changes:
  git add  OR git add .
- Commit Changes:
  git commit -m "commit message"
- Push to Remote Repository:
  git push origin 
- Pull Changes:
  git pull origin 
- View Log:
  git log

#### 2. **CI/CD (Jenkins, GitLab, etc.)**
- Jenkins CLI Trigger:
  java -jar jenkins-cli.jar -s  build 
- GitLab CI Lint:
  gitlab-runner verify
- CircleCI Config Validation:
  circleci config validate
- YAML Linting:
  yamllint 

#### 3. **Containerization (Docker, Podman)**
- Build Docker Image:
  docker build -t  .
- Run Container:
  docker run -d -p : 
- List Running Containers:
  docker ps
- Stop Container:
  docker stop 
- Podman Run:
  podman run -d -p : 

#### 4. **Container Orchestration (Kubernetes)**
- Check Nodes:
  kubectl get nodes
- Apply Configuration:
  kubectl apply -f 
- Delete Resources:
  kubectl delete -f 
- View Pod Logs:
  kubectl logs 
- Check Deployments:
  kubectl get deployments

#### 5. **Infrastructure as Code (Terraform, CloudFormation)**
- Terraform Initialization:
  terraform init
- Validate Terraform Files:
  terraform validate
- Plan Infrastructure:
  terraform plan
- Apply Changes:
  terraform apply
- Validate CloudFormation Template:
  aws cloudformation validate-template --template-body file://

#### 6. **Configuration Management (Ansible, Puppet, Chef)**
- Run Ansible Playbook:
  ansible-playbook 
- Lint Ansible Playbook:
  ansible-lint 
- Puppet Syntax Check:
  puppet parser validate 
- Chef Cookstyle Check:
  chef exec cookstyle

#### 7. **Monitoring and Logging (Prometheus, ELK, etc.)**
- Prometheus Configuration Check:
  promtool check config /path/to/prometheus.yml
- Elasticsearch Test Query:
  curl -X GET "localhost:9200/_cat/indices?v"
- Grafana Logs (Systemd):
  sudo journalctl -u grafana-server

#### 8. **Cloud Platforms (AWS, Azure, GCP)**
- AWS CLI Test:
  aws s3 ls
- Azure CLI Test:
  az account show
- GCP CLI Test:
  gcloud projects list

#### 9. **Secrets Management (Vault, AWS Secrets Manager)**
- Check Vault Status:
  vault status
- Retrieve AWS Secret:
  aws secretsmanager get-secret-value --secret-id 

#### 10. **General Commands**
- Check Open Ports:
  netstat -tuln
- Test Network Connectivity:
  ping 
- Curl Test:
  curl -I 
- Check Disk Usage:
  df -h
- Monitor System Processes:
  top OR htop
