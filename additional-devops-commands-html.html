---

### **1. Version Control (Git)**
- View Difference Before Commit:
  ```bash
  git diff
  ```
- Stash Changes:
  ```bash
  git stash
  ```
- Apply Stashed Changes:
  ```bash
  git stash apply
  ```
- Revert Commit:
  ```bash
  git revert 
  ```
- Merge Branches:
  ```bash
  git merge 
  ```

---

### **2. CI/CD**
- Test CI/CD Pipeline Locally:
  - **GitHub Actions**:
    ```bash
    act
    ```
  - **Jenkinsfile** (Pipeline Debugging):
    Use Jenkins Blue Ocean plugin for visualization.

---

### **3. Containerization (Docker, Podman)**
- Prune Unused Resources:
  ```bash
  docker system prune
  ```
- Check Container Resource Usage:
  ```bash
  docker stats
  ```
- Export/Import Images:
  ```bash
  docker save  | gzip > image.tar.gz
  docker load < image.tar.gz
  ```
- Podman Check Running Containers:
  ```bash
  podman ps
  ```

---

### **4. Kubernetes**
- Describe Pod:
  ```bash
  kubectl describe pod 
  ```
- Check Service Details:
  ```bash
  kubectl get svc
  ```
- Port Forward a Pod:
  ```bash
  kubectl port-forward pod/ :
  ```
- Debug Container in a Pod:
  ```bash
  kubectl exec -it  -- /bin/sh
  ```

---

### **5. Infrastructure as Code**
- **Terraform**:
  - Show State:
    ```bash
    terraform show
    ```
  - Destroy Resources:
    ```bash
    terraform destroy
    ```
- **CloudFormation**:
  - Delete Stack:
    ```bash
    aws cloudformation delete-stack --stack-name 
    ```

---

### **6. Configuration Management**
- **Ansible**:
  - Check Host Connectivity:
    ```bash
    ansible all -m ping -i 
    ```
- **SaltStack**:
  - Run a Test Command on All Minions:
    ```bash
    salt '*' test.ping
    ```

---

### **7. Monitoring and Logging**
- **Prometheus**:
  - Reload Configuration Without Restarting:
    ```bash
    curl -X POST http://:9090/-/reload
    ```
- **Elasticsearch**:
  - Check Cluster Health:
    ```bash
    curl -X GET "localhost:9200/_cluster/health?pretty"
    ```

---

### **8. Cloud Platforms**
- **AWS**:
  - List EC2 Instances:
    ```bash
    aws ec2 describe-instances
    ```
  - List S3 Buckets:
    ```bash
    aws s3 ls
    ```
- **Azure**:
  - List Resource Groups:
    ```bash
    az group list
    ```
  - View Current Subscription:
    ```bash
    az account show
    ```

---

### **9. Secrets Management**
- **Vault**:
  - Login to Vault:
    ```bash
    vault login 
    ```
  - Write a Secret:
    ```bash
    vault kv put secret/ =
    ```

---

### **10. Linux/General Commands**
- Search for a Process:
  ```bash
  ps aux | grep 
  ```
- Check Memory Usage:
  ```bash
  free -h
  ```
- Monitor Network Usage:
  ```bash
  iftop OR nload
  ```
- Check System Uptime:
  ```bash
  uptime
  ```
