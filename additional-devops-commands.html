Additional DevOps Commands

← Back to Documentation Index

1. Version Control (Git)

  • View Difference Before Commit:
    git diff
  • Stash Changes:
    git stash
  • Apply Stashed Changes:
    git stash apply
  • Revert Commit:
    git revert <commit_hash>
  • Merge Branches:
    git merge <branch_name>

2. CI/CD

  • Test CI/CD Pipeline Locally:
    GitHub Actions:
    act
    Jenkinsfile (Pipeline Debugging):
    Use Jenkins Blue Ocean plugin for visualization.

3. Containerization (Docker, Podman)

  • Prune Unused Resources:
    docker system prune
  • Check Container Resource Usage:
    docker stats
  • Export/Import Images:
    docker save <image_name> | gzip > image.tar.gz
    docker load < image.tar.gz
  • Podman Check Running Containers:
    podman ps

4. Kubernetes

  • Describe Pod:
    kubectl describe pod <pod_name>
  • Check Service Details:
    kubectl get svc
  • Port Forward a Pod:
    kubectl port-forward pod/<pod_name> <local_port>:<container_port>
  • Debug Container in a Pod:
    kubectl exec -it <pod_name> -- /bin/sh

5. Infrastructure as Code

  • Terraform:
  • Show State:
    terraform show
  • Destroy Resources:
    terraform destroy
  • CloudFormation:
  • Delete Stack:
    aws cloudformation delete-stack --stack-name <stack_name>

    • 6. Configuration Management

    • Ansible:
    • Check Host Connectivity:
      ansible all -m ping -i <inventory_file>
  • SaltStack:
  • Run a Test Command on All Minions:
    salt '*' test.ping

    • 7. Monitoring and Logging

    • Prometheus:
    • Reload Configuration Without Restarting:
      curl -X POST http://<prometheus_url>:9090/-/reload
  • Elasticsearch:
  • Check Cluster Health:
    curl -X GET "localhost:9200/_cluster/health?pretty"

    • 8. Cloud Platforms

    • AWS:
    • List EC2 Instances:
      aws ec2 describe-instances
    • List S3 Buckets:
      aws s3 ls
  • Azure:
  • List Resource Groups:
    az group list
  • View Current Subscription:
    az account show

    • 9. Secrets Management

    • Vault:
    • Login to Vault:
      vault login <token>
    • Write a Secret:
      vault kv put secret/<key> <field>=<value>

    10. Linux/General Commands

    • Search for a Process:
      ps aux | grep <process_name>
    • Check Memory Usage:
      free -h
    • Monitor Network Usage:
      iftop OR nload
    • Check System Uptime:
      uptime
