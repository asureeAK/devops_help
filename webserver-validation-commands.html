Web Server Configuration Validation Commands

1. Nginx

  • Validate Configuration:
    sudo nginx -t
  • Reload Configuration:
    sudo systemctl reload nginx
  • Debug Logs:
    sudo tail -f /var/log/nginx/error.log

Application Language Validation Commands

1. PHP

  • Syntax Check for a File:
    php -l /path/to/file.php
  • Debug Errors:
    error_reporting(E_ALL);
    ini_set('display_errors', 1);
  • Verify Installed PHP Version:
    php -v

General Functional Verification Commands

  • Ping the Server:
    ping <server-ip-or-domain>
  • Test with CURL:
    curl -I http://<server-ip-or-domain>
    Expected response: HTTP 200 OK
  • Check Open Ports:
    sudo netstat -tuln | grep <port-number>
  • Verify DNS Resolution:
    nslookup <domain>
  • Firewall Status:
    sudo ufw status
